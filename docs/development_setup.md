# Development setup

### Bazel

```bash
brew instal bazelisk
```

### Direnv

```bash
brew install direnv
```

```bash
echo 'eval "$(direnv hook zsh)"' >> ~/.zshrc
```

```bash
exec zsh
```

```bash
direnv allow
```

### Pre-commit hooks

```bash
brew install prek
```

```bash
prek install
```

### VSCode

```bash
bin/install-coppice-vscode-dev-extension
```

Then: `Developer: Reload Window`.
